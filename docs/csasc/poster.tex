\section*{1. Introduction \& Objectives}
\color{TechGrey}

\noindent
Pneumonia remains a leading cause of global mortality. While Deep Learning models like \textbf{ResNet} have revolutionized diagnostics \cite{He2016}, they face efficiency limits. This project investigates \textbf{Quantum Machine Learning (QML)} as a solution for the \textbf{NISQ} (Noisy Intermediate-Scale Quantum) era \cite{Schuld2018}.

\vspace{0.5em}
\textbf{Main Objectives:}
\begin{itemize}
    \item \textbf{\textcolor{UWBBlue}{Hybrid Architecture:}} Combine a classical CNN (feature extractor) with a Variational Quantum Classifier (VQC).
    \item \textbf{\textcolor{UWBBlue}{Efficiency:}} Test if quantum layers can reduce parameters while maintaining accuracy.
    \item \textbf{\textcolor{UWBBlue}{Benchmarking:}} Compare against classical methods on Chest X-Ray data \cite{Kundu2021}.
\end{itemize}

\color{UWBBlue}
\section*{2. Methodology: The Hybrid Pipeline}
\color{TechGrey}

\noindent
To address current hardware constraints, we designed a multi-stage pipeline:

1.  \textbf{Classical Feature Extraction:} Input X-ray ($224 \times 224$) processed by \textbf{ResNet-50} (headless) to extract a 2048-dimensional feature vector.
    
2.  \textbf{Dimensionality Reduction (PCA):} Compressed to \textbf{64 components} to fit the 6-qubit system ($2^6 = 64$) \cite{SharmaEnc2023}.

3.  \textbf{Quantum Classification (VQC):} Data is mapped to quantum states via \textbf{Amplitude Encoding} and processed by a parameterized circuit (Ansatz).

\color{UWBBlue}
\section*{3. Mathematical Formulation}
\color{TechGrey}

\noindent
\textbf{Amplitude Encoding:} The feature vector $x \in \mathbb{R}^{64}$ is mapped to the amplitudes of a 6-qubit state $|\psi\rangle$:
\begin{equation}
    |\psi(x)\rangle = \sum_{i=0}^{63} x_i |i\rangle \quad \text{where} \quad \sum |x_i|^2 = 1
\end{equation}

\noindent
\textbf{VQC Prediction:} The classification score is the expectation value of the Pauli-Z operator after applying the unitary ansatz $U(\theta)$ \cite{Mitarai2018}:
\begin{equation}
    f(x, \theta) = \langle \psi_x | U^\dagger(\theta) Z_0 U(\theta) | \psi_x \rangle
\end{equation}

\color{UWBBlue}
\section*{5. Results: Diagnostic Performance}
\color{TechGrey}

\noindent
Tested on 624 images (Guangzhou dataset). We observed a trade-off between efficiency and raw accuracy, partly due to the \textbf{Barren Plateaus} problem common in QML \cite{McClean2018}.

\begin{itemize}
    \item \textbf{Sensitivity (Recall):} \textbf{80.00\%} – Successfully detects most positive cases.
    \item \textbf{Accuracy:} 71.63\% (Hybrid) vs 85.90\% (Classical).
    \item \textbf{Specificity:} 57.69\% – impacted by \textbf{Dataset Shift} (Train: 74\% Pneumonia vs Test: 62\%) \cite{Zech2018}.
\end{itemize}

\color{UWBBlue}
\section*{6. Conclusion}
\color{TechGrey}

\noindent
This thesis demonstrated a functional hybrid pipeline. While the classical ResNet remains superior in accuracy, the Hybrid QML model achieved viable results (AUC 0.74) with \textbf{1527x fewer parameters}, proving the potential of quantum circuits for memory-constrained medical applications.
