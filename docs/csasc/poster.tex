\color{UWBBlue}\section*{1. Introduction \& Scientific Context}
\color{TechGrey}

\noindent
\textbf{Clinical motivation.} Pneumonia causes \(\sim\)2.5 million deaths annually. Manual radiologist interpretation is subjective; deep learning (ResNet, DenseNet) achieves \(>85\%\) diagnostic accuracy but requires massive parameter counts (164.8k+ per decision layer).

\vspace{0.25cm}
\textbf{Quantum computing opportunity.} NISQ devices (6--100 qubits) offer parameter-efficient learning via Variational Quantum Algorithms (VQA). However, their effectiveness in medical imaging remains underexplored, especially hybrid architectures combining classical feature extraction with quantum classification \cite{Schuld2018, Cerezo2021}.

\vspace{0.25cm}
\begin{tcolorbox}
\textbf{\textcolor{NTCPurple}{Research hypothesis:}}
A quantum decision layer (VQC with \(\sim\)100 parameters) can approach classical discrimination while reducing memory footprint, enabling edge/IoT deployment.
\end{tcolorbox}

%----------------------------------------------------------------------------------------
% 2. RELATED WORK
%----------------------------------------------------------------------------------------
\color{UWBBlue}\section*{2. Literature Review}
\color{TechGrey}

\noindent
Recent advances in QML and medical imaging:
\begin{itemize}[leftmargin=0.9em,itemsep=0.15em,topsep=0.1em]
  \item \textbf{Classical baseline:} ResNet for image recognition \cite{He2016}.
  \item \textbf{Quantum encoding:} encoding strategies and trade-offs \cite{SharmaEnc2023}.
  \item \textbf{VQC foundations:} quantum circuit learning \cite{Mitarai2018}.
  \item \textbf{Barren plateaus:} vanishing gradients in PQCs \cite{McClean2018}.
  \item \textbf{Dataset shift:} pneumonia generalization issues across sites \cite{Zech2018}.
\end{itemize}

\noindent
\textbf{Gap:} Limited empirical studies quantifying the efficiency--accuracy trade-off of CNN$\rightarrow$VQC hybrids under realistic distribution shifts.

\color{TechGrey}

\noindent
\textbf{Design rationale.} We separate \textit{feature learning} (classical) from \textit{decision making} (quantum) to stay within NISQ constraints.

\textbf{Feature extraction:} ResNet-50 without classification head outputs a 2048-D feature vector. \\
\textbf{PCA compression:} 2048 \(\rightarrow\) 64 dimensions to match 6-qubit amplitude encoding. \\
\textbf{Quantum classifier:} 6-qubit VQC with a shallow ansatz and 108 trainable parameters.

%----------------------------------------------------------------------------------------
% 4. MATHEMATICAL FORMULATION
%----------------------------------------------------------------------------------------
\color{UWBBlue}\section*{4. Quantum Circuit: Theory}
\color{TechGrey}

\noindent
\textbf{Amplitude encoding:} normalized vector \(x \in \mathbb{R}^{64}\), \(\|x\|=1\), mapped to:
\begin{equation}
|\psi(x)\rangle = \sum_{i=0}^{63} x_i |i\rangle \in \mathcal{H}_{2^6}
\end{equation}

\noindent
\textbf{VQC prediction:} with ansatz \(U(\theta)\) and observable \(Z_0\):
\begin{equation}
f(x,\theta) = \langle \psi(x) | U^\dagger(\theta)\, Z_0\, U(\theta) | \psi(x) \rangle \in [-1,1]
\end{equation}

\noindent
Probability mapping: \(\sigma(f)=(f+1)/2\). \\
Loss: binary cross-entropy, optimized via parameter-shift gradients \cite{Mitarai2018}.

\color{UWBBlue}\section*{5. Experimental Setup}
\color{TechGrey}

\noindent
\textbf{Dataset:} Guangzhou Chest X-ray dataset (624 images: 312 pneumonia, 312 normal).

\begin{itemize}[leftmargin=0.9em,itemsep=0.1em,topsep=0.1em]
  \item \textbf{Train/Test:} 70/30 split (437 train, 187 test).
  \item \textbf{Shift:} Train 74\% pneumonia vs Test 62\% pneumonia (robustness stress test).
\end{itemize}

\noindent
\textbf{Hyperparameters:}
\begin{itemize}[leftmargin=0.9em,itemsep=0.1em,topsep=0.1em]
  \item Optimizer: Adam (lr = 0.01, \(\beta_1=0.9\), \(\beta_2=0.999\))
  \item Epochs: 100, batch size: 16
  \item PCA dims: 64 (ablation: 8, 16, 32, 64, 128)
  \item Circuit depth: 4 layers
\end{itemize}

\noindent
\textbf{Simulation:} IBM Qiskit (statevector simulator, noise-free).

%----------------------------------------------------------------------------------------
% 9. LIMITATIONS & MITIGATION
%----------------------------------------------------------------------------------------
\color{UWBBlue}\section*{9. Limitations \& Discussion}
\color{TechGrey}

\noindent
\textbf{1) Barren plateaus.} Gradients can vanish in PQCs; mitigation includes ansatz design and training strategies \cite{McClean2018, Cerezo2021}.\\
\textbf{2) Dataset shift.} Prevalence differences can reduce specificity; mitigation includes calibration/domain adaptation \cite{Zech2018}.\\
\textbf{3) Hardware noise.} Real devices have decoherence and gate errors; mitigation includes error mitigation techniques.\\
\textbf{4) PCA loss.} Compression may discard useful variance; alternative encodings could be explored \cite{SharmaEnc2023}.

%----------------------------------------------------------------------------------------
% 10. CONCLUSION
%----------------------------------------------------------------------------------------
\color{UWBBlue}\section*{10. Conclusion \& Future Work}
\color{TechGrey}

\noindent
This study demonstrates a functional hybrid quantum--classical pipeline for pneumonia detection. The VQC decision layer achieves \textbf{AUC 0.74 with 1527\(\times\) parameter reduction}, supporting the case for parameter-efficient quantum decision heads in constrained settings.

\vspace{0.15cm}
\textbf{Future directions:}
\begin{itemize}[leftmargin=0.9em,itemsep=0.1em,topsep=0.1em]
  \item Run on real hardware with error mitigation.
  \item Evaluate on larger datasets (CheXpert / NIH).
  \item Improve robustness to dataset shift (calibration, balanced training).
\end{itemize}
